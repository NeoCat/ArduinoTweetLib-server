Tweet Library for Arduino

Post messages to Twitter (tweet) from Aruduino with Ethernet Shield!

How to begin:

Step 1 : Get a token to post a message using OAuth.

Step 2 : Add some Libraries to your Arduino IDE.

  • for Arduino IDE 0019 or later
    • Download the Arduino Tweet Library and put Twitter folder into ~/Documents/Arduino/libraries (Mac) or My Documents\Arduino\libraries\ (Windows). If there is no such folder, please create it.
    • Download the EthernetDNS Library and install it.
      • EthernetDNS library in this archive is patched to work with Udp library in IDE 0019 or later. Original version (distributed at gkaindl.com) currently does not support IDE 0019 or later.
  • for Arduino IDE 0018 or earlier
    • Download the Arduino Tweet Library and put Twitter folder into ~/Documents/Arduino/libraries (Mac) or My Documents\Arduino\libraries\ (Windows). If there is no such folder, please create it.
    • Download the Arduino Ethernet Library and install EthernetDNS and EthernetDHCP.
      • EthernetDNS library in this archive is patched to avoid an error by CNAME entry in the DNS response. Twitter library won't work without this patch, so please replace it if you already have the library. Original version is distributed at gkaindl.com.
  • Re-launch the IDE.
  • Detailed information about libraries installation.

Step 3 : Run a sample sketch to tweet!

You can get this sketch from File menu > Examples > Twitter > SimplePost .
Don't forget to add your token into the sketch.
Notice: This example is for IDE 0019 or later. Notice: the library uses this site as a proxy server for OAuth stuff. Your tweet may not be applied during maintenance of this site. 
#include <SPI.h>
#include <Ethernet.h>
#include <UDP.h>
#include <EthernetDNS.h>
#include <Twitter.h>

byte mac[] = { 0xDE, 0xAD, 0xBE, 0xEF, 0xFE, 0xED };
byte ip[] = { 10, 0, 0, 177 };
Twitter twitter("<<< your token here >>>");
char msg[] = "Hello, World! I'm Arduino!";

void setup()
{
  delay(1000);
  Ethernet.begin(mac, ip);
  Serial.begin(9600);
  
  Serial.println("connecting ...");
  if (twitter.post(msg)) {
    int status = twitter.wait();
    if (status == 200) {
      Serial.println("OK.");
    } else {
      Serial.print("failed : code ");
      Serial.println(status);
    }
  } else {
    Serial.println("connection failed.");
  }
}

void loop()
{
}
